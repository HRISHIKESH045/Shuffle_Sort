# Shuffle Sort Game

A simple web-based game to shuffle and sort numbers.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Users can shuffle numbers randomly.
- Users can sort numbers in ascending order.
- Responsive design that works on desktop and mobile browsers
